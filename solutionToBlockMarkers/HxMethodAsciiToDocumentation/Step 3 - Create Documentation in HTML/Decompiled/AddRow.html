AddRowToArrayTable

Description

Adds a row to the active table using default formatting

*Markdown Support*: Text in i_arrStrEntries[] supports inline formatting:

  • **bold text** : Renders text in bold
  • *italic text* : Renders text in italic
  • ***bold+italic*** : Renders text in bold and italic

Return Type

Boolean

List of Parameters

Name Direction Type Description
i_arrStrEntriesInArray of Variables[in] Array Input of String Entries
i_intPaddingTopInVariable[in] Input of Integer Padding Top
i_intPaddingBottomInVariable[in] Input of Integer Padding Bottom
