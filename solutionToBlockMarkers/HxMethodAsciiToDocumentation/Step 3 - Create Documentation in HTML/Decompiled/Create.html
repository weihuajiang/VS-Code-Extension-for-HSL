Create

Description

Create creates a new Array Table. The resulting array contains: description, sizeX, sizeY, then the data region, and a column-name block.

Return Type

No return variable

List of Parameters

Name Direction Type Description
i_strDescription In Variable Description string stored in the table header (index 0).
i_intSizeX In Variable Number of columns (X dimension). Must be >= 1.
i_intSizeY In Variable Number of rows (Y dimension). Must be >= 0.
i_arrColumnNames In Array of Variables List of column names. Must equal i_intSizeX.
o_arrValues In/Out Array of Variables The created Array Table (header + data region + column-name block).
