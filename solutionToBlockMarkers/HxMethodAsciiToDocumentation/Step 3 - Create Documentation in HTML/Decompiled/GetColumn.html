GetColumn

Description

 Get all values of a column (all values of the y dimension)
     
 Throws an exception if
 - input is not a  Array Table 
 - input has wrong type
 - input is out of range
 - column names are not set

Return Type

No return variable

List of Parameters

Name Direction Type Description
i_arrValues In Array of Variables Array Table to get the colum name from.
i_intPositionX In Variable Position in x dimension (1..x)
o_arrValues In/Out Array of Variables Array containing column values
o_varColumnName In/Out Variable Column name out.
