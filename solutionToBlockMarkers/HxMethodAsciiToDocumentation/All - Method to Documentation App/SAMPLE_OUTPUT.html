TransferSample

Description

This submethod transfers samples from the source plate to the destination plate using the specified pipetting channels. The method handles liquid aspiration and dispensing with configurable volumes and speeds.

Parameters

Parameter Name Description
i_strSourceLabwareID String identifier for the source labware position on the deck
i_strDestLabwareID String identifier for the destination labware position on the deck
i_intChannelPattern Integer pattern defining which pipetting channels to use (bit mask)
i_fltVolume Float value specifying the volume to transfer in microliters
i_blnMixAfterDispense Boolean flag to enable mixing after dispensing the sample
o_intErrorCode Integer output code indicating success (0) or error condition
