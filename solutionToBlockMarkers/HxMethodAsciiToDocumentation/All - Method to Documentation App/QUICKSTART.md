# Quick Start Guide

## Get Started in 2 Steps!

### Step 1: Install pywin32 (One-time only)

```powershell
pip install pywin32
```

**Important**: You must restart your IDE or terminal after this command.

### Step 2: Run the Application

```powershell
python MethodToDocumentation.py
```

Or double-click `MethodToDocumentation.py` in Windows Explorer.

## Using the Application

1. Click "Browse..." next to ".MED File" and select your method file
2. Click "Browse..." next to "Output Directory" and choose where to save HTML files
3. (Optional) Enter your name in "Author Name"
4. Click "Generate Documentation"
5. Open the HTML files in your web browser!

## That's It!

Your HTML documentation is ready. Each submethod has its own HTML file with:
- Method description
- Parameter table
- Professional styling
